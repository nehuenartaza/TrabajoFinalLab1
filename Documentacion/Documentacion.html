Astronautas

Función que realiza operaciones relacionadas con los astronautas.

  • registerAstronaut()

    Registra un astronauta.

  • saveAstronaut(stAstronaut user)

    Guarda un astronauta en un archivo.

  • confirmAstronautID(int ID)

    Usada para retornar un flag=1 si la ID enviada se encontró en un registro.

  • changeAstronautNickname()

    Cambia el apodo de un astronauta.

  • changeAstronautAge()

    Cambia la edad de un astronauta.

  • changeAstronautSpeciality()

    Cambia la especialidad de un astronauta.

  • changeAstronautFlightTime()

    Cambia las horas de vuelo de un astronauta.

  • changeAstronautHoursSpaceStation()

    Cambia las horas que estuvo un astronauta en la estación espacial.

  • changeAstronautAmmountMissions()

    Cambia la cantidad de misiones en las que participó un astronauta.

  • changeAstronautStatus()

    Cambia el estado de un astronauta, 1-Activo, 2-Retirado.

  • selectAstronaut()

    Selecciona un astronauta y retorna la posición del mismo.

  • showAstronauts()

    Muestra la totalidad del contenido del archivo de astronautas.

  • astronautAvailability(int ID)

    Retorna la disponibilidad del astronauta, 1-activo, 2-retirado, NULL-no existe.

  • getLastAstronautID()

    Toma la última ID de astronauta registrada y la retorna.

  • getTotalAstronauts()

    Retorna la cantidad total de registros de astronautas.

  • firstAstronaut()

    Verifica si al menos hay un astronauta registrado.

  • showAstronautByID()

    Muestra un astronauta con el ID especificado.

  • showActiveAstronauts()

    Muestra los astronautas activos.

  • showRetiredAstronauts()

    Muestra los astronautas retirados.

  • searchAstronautByName()

    Busca un astronauta por nombre y muestra su información.

Otras funciones de Modificacion

Estas funciones aplican

  • pprintf(const char *str)

    Esta función imprime una cadena de caracteres de manera gradual en la salida estándar. Si la función proceduralTextRead() devuelve verdadero, la cadena se imprime carácter por carácter con una pausa de 1000 microsegundos entre cada uno. Después de imprimir la cadena completa, se realiza una pausa adicional de 10000 microsegundos. Si la función proceduralTextRead() devuelve falso, la cadena se imprime utilizando la función printf() de forma normal.

  • changeColor()

    Esta función permite al usuario cambiar el color del texto en la consola. Primero muestra las opciones de colores llamando a la función textColorsOptions(). Luego, solicita al usuario que ingrese un número de opción y llama a la función colorOption() para establecer el color seleccionado. El ciclo do-while continúa hasta que el usuario ingrese una opción válida. Finalmente, guarda el cambio de color llamando a la función saveColorChange().

  • setColor(int colorCode)

    Esta función establece el color de texto en la consola utilizando el código de color proporcionado. Recibe el parámetro colorCode que representa el código de color deseado y utiliza las funciones de la biblioteca de Windows para establecer el color.

  • textColorsOptions()

    Esta función muestra las opciones de colores disponibles al usuario. Utiliza la función pprintf() para imprimir gradualmente las opciones de color en la consola, utilizando diferentes colores para resaltar cada opción. Luego llama a la función readUserData() para leer los datos del usuario.

  • colorOption(int option)

    Esta función establece el color de texto en la consola según la opción seleccionada por el usuario. Recibe el parámetro option que representa la opción de color elegida y utiliza una declaración switch para asignar el código de color correspondiente.

  • saveColorChange(int colorChosen)

    Esta función guarda el cambio de color seleccionado por el usuario. Crea una estructura stUserData que almacena el valor de proceduralTextRead() y colorChosen. Abre un archivo para escritura binaria, escribe los datos de la estructura en el archivo y luego lo cierra. Si ocurre un error al abrir el archivo, muestra un mensaje de error.

  • readUserData()

    Esta función lee los datos del usuario almacenados en un archivo. Intenta abrir el archivo para lectura binaria. Si el archivo no existe, crea uno nuevo y escribe datos predeterminados en él. Si el archivo existe, lee los datos en una estructura stUserData, obtiene el color almacenado y llama a colorOption() para establecer ese color. Luego cierra el archivo.

  • proceduralTextRead()

    Esta función lee el valor de proceduralText almacenado en un archivo y lo devuelve. Abre el archivo para lectura binaria, lee el valor y luego cierra el archivo.
