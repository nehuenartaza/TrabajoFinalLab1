Documentacion oficial TP Laboratorio COM 1

Dependencias:

  • stdio.h: Proporciona las funciones básicas de entrada y salida estándar, como printf y scanf.
  • stdlib.h: Contiene funciones para realizar operaciones de asignación de memoria, control de procesos y otras utilidades.
  • string.h: Ofrece funciones para manipular cadenas de caracteres, como strcpy y strlen.
  • unistd.h: Proporciona funciones para el acceso a características específicas del sistema operativo, como la función sleep.
  • time.h: Contiene funciones para trabajar con fechas y tiempos.
  • stdbool.h: Define el tipo de dato bool y los valores true y false.
  • windows.h: Archivo de cabecera específico de Windows que contiene definiciones para la programación de Windows.

Constantes:

  • dimInt: Macro que define la dimensión de un array de enteros como 15.
  • dimChar: Macro que define la dimensión de un array de caracteres como 50.
  • Fastronauts: Constante que representa el nombre del archivo binario "astronautsData.bin".
  • Fstarships: Constante que representa el nombre del un archivo binario "starshipsData.bin".
  • Fmissions: Constante que representa el nombre del un archivo binario "missionsData.bin".
  • FuserData: Constante que representa el nombre del un archivo binario "userData.bin".

Otras funciones de Modificacion

Estas funciones rotan en torno al usuario y sus preferencias

  • pprintf(const char *str)

    Esta función imprime una cadena de caracteres gradualmente en la salida estándar. Si la función proceduralTextRead() devuelve verdadero, la cadena se imprime carácter por carácter con una pausa de 1000 microsegundos entre cada uno. Después de imprimir la cadena completa, se realiza una pausa adicional de 10000 microsegundos. Si la función proceduralTextRead() devuelve falso, la cadena se imprime utilizando la función printf() de forma normal.

  • inThere(int forSearch, int array[], int valids)

    Esta función verifica si un valor determinado está presente en un arreglo de enteros. Recibe el valor a buscar, el arreglo y la cantidad de elementos válidos en el arreglo. Itera sobre el arreglo y devuelve verdadero si encuentra el valor, y falso en caso contrario.

  • changeColor()

    Esta función permite al usuario cambiar el color del texto en la consola. Muestra las opciones de colores llamando a la función textColorsOptions(). Luego, solicita al usuario que ingrese un número de opción y llama a la función colorOption() para establecer el color seleccionado. El ciclo do-while continúa hasta que el usuario ingrese una opción válida. Finalmente, guarda el cambio de color llamando a la función saveColorChange().

  • setColor(int colorCode)

    Esta función establece el color de texto en la consola utilizando el código de color proporcionado. Utiliza funciones de la biblioteca de Windows para establecer el color.

  • textColorsOptions()

    Esta función muestra las opciones de colores disponibles al usuario. Utiliza la función pprintf() para imprimir gradualmente las opciones de color en la consola, utilizando diferentes colores para resaltar cada opción. Luego llama a la función readUserData() para leer los datos del usuario.

  • colorOption(int option)

    Esta función establece el color de texto en la consola según la opción seleccionada por el usuario. Recibe el parámetro option que representa la opción de color elegida y utiliza una declaración switch para asignar el código de color correspondiente.

  • saveColorChange(int colorChosen)

    Esta función guarda el cambio de color seleccionado por el usuario. Crea una estructura stUserData que almacena el valor de proceduralTextRead() y colorChosen. Abre un archivo para escritura binaria, escribe los datos en el archivo utilizando la función fwrite() y luego cierra el archivo. Si la operación de escritura es exitosa, muestra un mensaje de éxito utilizando la función pprintf(). Si no se puede abrir el archivo, muestra un mensaje de error.

  • readUserData()

    Esta función lee los datos del usuario desde un archivo. Primero intenta abrir el archivo en modo lectura binaria. Si el archivo no existe, lo crea en modo escritura binaria y escribe valores predeterminados en él. Si el archivo existe, lee los datos almacenados en la estructura stUserData utilizando la función fread(). Luego, obtiene el valor del color del usuario y llama a la función colorOption() para establecer el color. Finalmente, cierra el archivo.

  • proceduralTextRead()

    Esta función lee el valor de proceduralText desde un archivo y lo devuelve. Abre el archivo en modo lectura binaria, lee el valor almacenado en la estructura stUserData utilizando la función fread() y cierra el archivo. Devuelve el valor de proceduralText.

  • toggleProceduralText()

    Esta función cambia el estado de proceduralText en el archivo. Abre el archivo en modo lectura y escritura binaria. Lee los datos almacenados en la estructura stUserData utilizando la función fread(). Luego, cambia el estado de proceduralText y lo sobrescribe en el archivo utilizando la función fwrite(). Muestra un mensaje indicando si se activó o desactivó el "Procedural Text". Por último, cierra el archivo.
